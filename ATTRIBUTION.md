# Attribution

This project includes kanji readings, English meanings, vocabulary examples, and radical/component lists derived from Electronic Dictionary Research and Development Group data.

## EDRDG Data

Dictionary-derived content is based on:

- KANJIDIC2: https://www.edrdg.org/wiki/KANJIDIC_Project.html
- JMdict/EDICT: https://www.edrdg.org/wiki/JMdict-EDICT_Dictionary_Project.html
- KRADFILE/RADKFILE: https://www.edrdg.org/krad/kradinf.html
- EDRDG license: https://www.edrdg.org/edrdg/licence.html

EDRDG dictionary files are distributed under license terms maintained by EDRDG. The project planning notes for this dataset treat the derived content as CC BY-SA 4.0-compatible and require attribution in downstream app documentation.

When redistributing this dataset or generated JSON, keep attribution to EDRDG and preserve license compatibility for derived readings, meanings, vocabulary examples, and radical/component lists.

Suggested downstream attribution text:

```text
Kanji readings, English meanings, vocabulary examples, and radical/component lists are derived from KANJIDIC2, JMdict, and KRADFILE/RADKFILE data from the Electronic Dictionary Research and Development Group (EDRDG). See https://www.edrdg.org/ for source data and license terms.
```

## JLPT Level Placement

The JLPT does not publish official kanji lists for the modern N5-N1 levels after the 2010 exam change. The level placement in this repository is cross-checked against public study references and should be treated as study guidance, not an official exam specification.

Public level-reference pages used during Stage 1 preparation include:

- JLPT Sensei kanji lists: https://jlptsensei.com/jlpt-n5-kanji-list/
- KanjiQuest JLPT kanji list: https://kanjiquestblog.com/kanji-list-jlpt/
- Kanshudo JLPT kanji collection: https://www.kanshudo.com/collections/jlpt_kanji

Do not copy long explanations, lesson text, or proprietary example sentences from public JLPT study websites into this project. Use those sites only for level placement cross-checks.
